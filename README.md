# Personal Portfolio Website

A clean, responsive personal portfolio website built using vanilla HTML, CSS, and minimal JavaScript. This project serves as a central hub to showcase my background, education, technical skills, and recent development projects.

## 🚀 Features

- **About Me:** A brief introduction and profile card setup.
- **Education & Skills:** Structured sections highlighting academic background and core technical competencies.
- **Projects Showcase:** Clean cards summarizing recent technical projects.
- **Contact Form:** A stylized, user-friendly frontend contact form setup.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing windows.

## 🛠️ Built With

- **HTML5** - Semantic markup structure
- **CSS3** - Modern layouts using Flexbox, custom CSS properties/variables, and media queries
- **JavaScript** - Client-side interaction tracking
